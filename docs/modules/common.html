drawBorder (background, foreground, x, y, w, h, dialog, large, squareEdges)
Draws a fancy rounded (or not) border.

Parameters:

  • background The background color of the border.
  • foreground The foreground color of the border.
  • x The X position of the border.
  • y The Y position of the border.
  • w The width of the border. This should be at least 3.
  • h The height of the border. This should be at least 3.
  • dialog If true, this moves the border's bottom up one subpixel. This is used by the dialog element.
  • large Makes the border a little larger. This is used for inputs.
  • squareEdges If true, the rounded corners of the UIs will be disabled, and will be square instead.
textOverflow (text, max)
A function that will chop off text if it's longer than a specified number.

Parameters:

  • text The text to check for overflow.
  • max The maximum size of the text.
