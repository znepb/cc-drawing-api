Module button

A button api that also draws a fancy border around the button, giving it a rounded effect.

Info:

Functions

buttonApi.renderSingle (id, clicking)
Renders a single button.

Parameters:

  • id The ID of the button to render.
  • clicking Whether or not the event cycle is in between down and up.

See also:

    render
buttonApi.enableButton (id)
Enables a button.

Parameters:

  • id The ID of the button to enable.
buttonApi.disableButton (id)
Disables a button.

Parameters:

  • id The ID of the button to disable.
buttonApi.add (id, text, x, y, theme)
Creates a new button.

Parameters:

  • id The ID of the button. This will be the second parameter of a button_click event.
  • text The text that will be displayed on the button.
  • x The X position of the button. Note, this is the left corner of the button, not the text.
  • y The Y position of the button. Same goes for this as for the x parameter.
  • theme A theme table.
