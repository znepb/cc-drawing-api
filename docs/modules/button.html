Module button

buttonApi.renderSingle (id, borderOverride)
Renders a single button.

Parameters:

  • id The ID of the button to render.
  • borderOverride The border color of the button. Default is to inherit from the button's background.

buttonApi.add (id, text, x, y, bgc, fgc, clickColor, borderColor)
Creates a new button.

Parameters:

  • id The ID of the button. This will be the second parameter of a `button_click` event.
  • text The text that will be displayed on the button.
  • x The X position of the button. Note, this is the left corner of the button, not the text.
  • y The Y position of the button. Same goes for this as for the x parameter.
  • bgc The background color of the button. Default color is white.
  • fgc The foreground (text) color of the button. Default color is black.
  • clickColor The highlight color of the button's border. If not supplied, this will pull from the clickColors table.
  • borderColor The color of the whitespace around the button. THe default is to inherit this when the button is rendered.
