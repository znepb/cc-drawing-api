dialogApi.create (title, subtext, button, scroll, width, height, backgroundColor, borderColor, textColor, titleColor, highlightColor)
Create a dialog element, and render it.

Parameters:

  • title The title of the dialog UI.
  • subtext The description of the dialog.
  • button The button worker that the UI's OK button will be put in.
  • scroll The scrollbox instance that will be used for the textbox.
  • width Width of the dialog. Default of 26.
  • height Height of the dialog. Default of 10.
  • backgroundColor Background color of the dialog. Default color is white.
  • borderColor Color of the border and the area where the button is contained. Default color is light gray.
  • textColor Color of the subtext in the dialog. Default color is gray.
  • titleColor Color of the title in the dialog. Default color is black.
  • highlightColor Color of the button's border between mouse_click and mouse_up. Default color is gray.

Returns:

    A number which is the the second half the the button's ID. The way to find the ID of the button is `"ok-" .. id`. This is used to run a function when the OK button is pressed.
dialogApi.dismiss (id, button, scroll)
Dismisses a dialog box.

Parameters:

  • id The ID of the dialog, retured at creation.
  • button The button manager used at creation.
  • scroll The scroll manager used at creation.
