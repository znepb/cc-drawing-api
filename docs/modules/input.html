read (_sReplaceChar, _tHistory, _fnComplete, _sDefault, _nLimit)
Basicly, CC's read function with a custom width parameter.

Parameters:

  • _sReplaceChar The character to hide all the characters.
  • _tHistory A table containing a history for the textbox.
  • _fnComplete A function that is random when the text is completed. The only parameter is the line.
  • _sDefault The default text.
  • _nLimit The size limit of the read window.
render (id)
Renders a textbox.

Parameters:

  • id The ID of the textbox to render.
create (id, x, y, w, theme, placeholder, history, replace, default)
Creates a new input box.

Parameters:

  • id The ID of the text box. Used to remove, render, etc.
  • x The X position of the textbox.
  • y The Y position of the textbox.
  • w The width of the textbox.
  • theme The theme to use/
  • placeholder The text that will be shown when no text has been entered. Note that this is overidden by default. The difference is that placeholder text does not persist when the textbox is selected.
  • history The history table that the read function will draw from.
  • replace The replace character, mainly for passwords.
  • default The default text that will be entered. For mor information on this, see the placeholder paramerer.

init (manager)
The event listener for the text box. When the text box has been finished, an event will be queued, entitled "textbox_complete". The first parameter is the ID of the textbox, and the second is the text entered into said textbox.

Parameters:

  • manager The event manager.
