Functions

create (id, x, y, w, h, parent)
Create a new scroll box.

Parameters:

  • id The ID of the scrollbox.
  • x The X position of the scrollbox.
  • y The Y position of the scrollbox.
  • w The width of the scrollbox.
  • h The height of the scrollbox.
  • parent The parent terminal of the scrollbox.

Returns:

    A terminal-like instance, with a couple more functions.
remove (id)
Removes a scrollbox's event listener. Notice this does not clear it, you will have to clear the screen to remove it.

Parameters:

  • id The ID of the scroll box.
init (manager)
Initalizes the event manager for the scrollbox.

Parameters:

  • manager The event manager.
