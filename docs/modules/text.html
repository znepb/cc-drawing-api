textApi.create (text, iX, iY, eX, eY, term)
Create a text box.

Parameters:

  • text Text to display in the box.
  • iX The starting X position of the text frame.
  • iY The starting Y position of the text frame.
  • eX The ending X of the text frame.
  • eY The ending Y of the text frame.
  • term The terminal the text will be written to.
