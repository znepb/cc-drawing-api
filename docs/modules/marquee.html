create (x, y, w, h, parent)
Creates a marquee frame.

Parameters:

  • x The X position of the marquee frame.
  • y The Y position of the marquee frame.
  • w The width of the marquee frame.
  • h The height of the marquee frame.
  • parent The parent term of the marquee window.

Returns:

    A 500x500 terminal which can be used like a normal terminal.
