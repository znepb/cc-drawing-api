Checks if a context menu exists.

Returns:

    Whether or not the context menu exists.
create (id, x, y, elements, theme, presist)
Creates a new context menu.

Parameters:

  • id The ID of the context menu. When a context menu item is selected, a event with the name context_menu_select will be dispatched, with a first argument being the context menu id, and the second being thr button id.
  • x The X position of the context menu.
  • y The Y position of the context menu.
  • elements The elements that will be put inside the context menu. See tests/contextMenu.lua for an example.
  • theme The theme to use for the context menu.
  • presist If true, the context menu will stick around after being clicked or clicked outside of.
init (manager)
Initalizes the context menu's event manager.

Parameters:

  • manager The event manager instance to use.
